Recursos PATAS (PAWS en inglés)

Los siguientes archivos de recursos están disponibles para ayudarle a comprender varios aspectos acreca de PATAS (PAWS en inglés).

Los primeros cuatro recursos son para ayudarle con editando los boradores de las gramáticas que se producen por PATAS.

  • Guía de referencia para XLingPaper para la edición del dispositivo de salida del programa PATAS mediante el editor de XML XMLmind. Este documento enumera una serie de tareas comunes de los usuarios que a menudo quieren hacer al editar la salida del programa PATAS en XLingPaper.

  • Guía breve de referencia para XLingPaper para la edición del dispositivo de salida del programa PATAS mediante el editor de XML XMLmind. Una versión más breve de la guía de referencia para imprimir.

  • Estructura de un ejemplo interlinearizado con el programa XLingPaper. Este es un diagrama mostrando la estructura de un ejemplo interlinearizado.

  • La página principal del web de XLingPaper. En esta página discutimos el formato del procesador y damos referencias a algunas herraminetas que puede ayudarle a editar el borrador a la gramática más facilmente.

Los últimos tres recursos tratan sobre el analizador PC-PATR y la teoría de la sintaxis que se llama "Rección y Ligamiento". (Nótese que PATAS no sigue completemente la teoría de la sintaxis llamada "Rección y Ligamiento". Sin embargo, se utiliza la estructura "X-Barra".)

  • Una discusión sobre cómo utilizar el analizador PC-PATR como modelo de la teoría de Rección y Ligamiento. Este documento muestra los resultados de un experimento del modelo de la sintaxis de inglés con PC-PATR y la teoría Rección y Ligamiento.

  • Una introducción a la teoría de Rección y Ligamiento Este documento presenta una introducción que es fácil de entender y paso a paso.

  • Un manual de referencia para PC-PATR Este documento es un guía del analizador de sintazis PC-PATR y está muy detallada.
